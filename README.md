# Jollof Rush Web Game

Playable browser game where you catch illustrated jollof ingredients and avoid burnt pots before time runs out.

## Run

Open `index.html` in a browser.

## How to play

- Move the bowl with your mouse, touch, or arrow keys.
- Catch rice, pepper, tomato, and chicken icons.
- Avoid burnt pots.
- Score as high as possible before the timer ends.
